# Laravel-FullStack-Application
Implementación de buenas prácticas con Laravel

Esta es una aplicación web funcional, donde podemos recibir mensajes vía e-mail, 
también implementa un CRUD de proyectos donde los permisos se asignan dependiendo 
el acceso del cliente, podemos paginar los resultados, pero para crear, actualizar
y eliminar proyectos será necesario iniciar sesión, donde autenticamos que el 
usuario tiene los permisos necesarios
